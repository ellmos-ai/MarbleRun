# Contributing to MarbleRun (llmauto)

Thanks for your interest in contributing!

## Getting Started

1. Fork and clone: `git clone https://github.com/YOUR_USERNAME/MarbleRun.git`
2. Install: `pip install -e .` or run directly: `python -m llmauto --help`
3. Requires: Python 3.10+, Claude Code CLI

## How to Contribute

- **Bug fixes**: Open an Issue or submit a PR directly
- **Chain templates**: New workflow patterns in `templates/`
- **Shutdown conditions**: New chain control logic
- **Documentation**: Always appreciated

### Good First Issues

Look for [`good first issue`](https://github.com/ellmos-ai/MarbleRun/labels/good%20first%20issue) labels.

## Guidelines

- Keep zero-dependency philosophy (stdlib only)
- Python 3.10+, type hints where practical
- One feature/fix per PR
- Test chains locally before submitting
